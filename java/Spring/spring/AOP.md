# 动态代理

在运行时增强对象方法的功能

# 重点概念

## Pointcut（切入点）

被增强的方法

## Advice（通知/增强）

用来 增强的方法

## Aspect（切面）

切点+增强

## Weaving（织入）

将切点与增强结合的过程

## Target(代理对象)

代理的目标对象

## Proxy(代理)

一个类被AOP织入增强后，就产生了一个结果代理类

## Joinpoint(连接点)

所谓连接点指哪些被拦截到的点。是所有被拦截的方法的集合

## Pointcut(切入点)

对Joinpoint方法增强的方法
